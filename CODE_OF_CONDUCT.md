# Code of Conduct

This project adopts the **[Contributor Covenant version 3.0](https://www.contributor-covenant.org/version/3/0/)** as its code of conduct. All participants in project spaces — issues, pull requests, discussions, and any related communication channels — are expected to follow it.

The full canonical text is permanently available at: <https://www.contributor-covenant.org/version/3/0/>

In summary, this project is committed to providing a welcoming, safe, and equitable environment for all participants, regardless of background or identity. Harassment, character attacks, stereotyping, sexualization, breaches of confidentiality, and endangering behavior are not tolerated. The Contributor Covenant document at the link above describes encouraged and restricted behaviors in detail, and outlines the enforcement ladder (warning → temporarily limited activities → temporary suspension → permanent ban) used to address violations.

## Reporting a violation

To report a possible violation of this Code of Conduct, send an email to:

> **`rmwarriner@icloud.com`**

Reports will be reviewed promptly and handled with discretion. Where possible, enforcement actions are carried out in private with the involved parties; community-wide communication may be part of a mutually agreed-upon resolution.

## Scope

This Code of Conduct applies within all project community spaces, and also when an individual is officially representing this project in public spaces (e.g., posting from an official account, presenting at an event as a project representative).
